
## 部署到Cloudflare Pages

### 步骤1：创建GitHub仓库

1. 在GitHub上创建一个新的仓库
2. 将本项目的文件上传到仓库中

### 步骤2：部署到Cloudflare Pages

1. 登录Cloudflare账户
2. 进入Cloudflare Pages控制台
3. 点击「创建项目」按钮
4. 选择「连接到Git」选项
5. 选择你的GitHub仓库
6. 配置构建设置：
   - 框架预设：`None`
   - 构建命令：`echo "Build completed"`
   - 构建输出目录：`/`
7. 点击「部署站点」按钮
8. 等待部署完成，获取部署后的URL

### 步骤3：自定义域名（可选）

如果需要使用自定义域名，可以在Cloudflare Pages设置中添加自定义域名并配置DNS记录。
